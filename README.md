# vswm
Very Small Window Manager

(Based on TinyWM)

## Commands:

Command | Action
--- | ---
`Mod+F1` | Raise Window Under Pointer
`Mod+F2` | New Xterm
`Mod+F3` | New App Launcher (using `menu.sh`, default is Rofi.)
`Mod+Mouse1+Drag` | Move Window
`Mod+Mouse3+Drag` | Resize Window
